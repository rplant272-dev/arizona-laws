СИТУАЦИОННЫЕ КОДЫ (CODE)

CODE 0 – Офицер ранен
CODE 1 – Офицер в бедственном положении
CODE 2 – Обычный вызов с низким приоритетом (без сирен и спецсигналов, соблюдая ПДД)
CODE 2 HIGH – Приоритетный вызов (также без сирен и спецсигналов, соблюдая ПДД)
CODE 3 – Срочный вызов (сирены, спецсигналы, игнорирование некоторых пунктов ПДД)
CODE 4 – Помощь не требуется
CODE 4 ADAM – Помощь не требуется в данный момент, но близлежащие офицеры должны быть готовы оказать помощь
CODE 5 – Найдены незаконно засаженные плантации наркотических веществ (указать местоположение и состояние плантации)
CODE 7 – Перерыв на обед
CODE 30 – Срабатывание "тихой" сигнализации на месте происшествия
CODE 30 RINGER – Срабатывание "громкой" сигнализации на месте происшествия
CODE 37 – Обнаружение угнанного транспортного средства (указать номер, описание авто, направление движения)
